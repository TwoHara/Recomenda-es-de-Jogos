# Guia de Recomendações

Este projeto é um mini-site feito para a atividade de HTML e CSS. Nele eu recomendo alguns jogos e mostro informações básicas sobre cada um. A ideia principal foi treinar o uso de textos, listas, imagens, links, tabelas e estilos com CSS.

## Tecnologias usadas

* **HTML5**
* **CSS3**

## O que o site contém

* Títulos, subtítulos e parágrafos
* Imagens relacionadas aos jogos
* Listas ordenadas e não ordenadas
* Links internos e externos
* Uma tabela final usando `rowspan` e `colspan`
* Estilos personalizados com CSS (cores, fonte, fundo e tabela zebrada)

## Como acessar

O site pode ser visto pelo GitHub Pages.
